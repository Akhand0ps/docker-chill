
Docker commands:


```

>docker ps -> to show all the running containers
>docker ps -a -> to show all the containers
>docker images -> shows all the images
>docker images -a -> shows all the images , even hidden one
>docker stop <continer-id> -> stops the container
>docker run -it -P <image> -> auto PORT mapping, it will map the EXPOSED port to random PORT on local machine
>docker ps -> shows all the running containers , add -a to see all running/stopped containers.
```
